The Markdown syntax doesn't provide native support for image captions, but you can use a `::: figure` block:

:::: div example

```markdown
::: figure | Image caption
![Alt text](/assets/images/image.png)
:::
```

::: figure | Image caption
![Alt text](/assets/images/image.png)
:::
