## Usage

### Installation

#### Add it to your project

Using your package manager, for example with `uv`:

```bash
uv add writeadoc --group docs
```

or `poetry`

```bash
poetry add writeadoc --group docs
```

#### Or install it by itself

Using `uv`:

```bash
uv pip install writeadoc
```

... or regular `pip`:

```bash
pip install writeadoc
```
